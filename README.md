Features Overview

Multi-Track NLA Mixer: 4-Channel animation blending with individual weight and speed controls.

Visual Modes: PBR (Physically Based Rendering), NPR Toon (Non-Photorealistic), and Wireframe overlay.

Holo-Scan: Real-time shader injection for holographic visual analysis.

Environment Engine: Built-in procedural "Cyber" skybox and HDRI/EXR import support.

Post-Processing: Integrated Unreal Bloom and color grading.
